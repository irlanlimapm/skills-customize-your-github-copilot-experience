
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game (Hangman) using Python. Practice strings, loops, conditionals, and user input handling.

## 📝 Tasks

### 🛠️ Implement the Hangman game

#### Description
Create a command-line Hangman game that randomly chooses a secret word and lets the player guess letters until they either discover the word or run out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses and reveal matching letters in the word (display progress like `_ a _ _ m _ n`)
- Track and display remaining incorrect attempts
- Prevent the same correct/incorrect guess from being counted multiple times
- End the game with a clear win or lose message and reveal the full word on loss

## 📚 Starter files

- `starter-code.py` — minimal scaffolding to start from (see folder).

## ⚙️ Setup & Run

Requires Python 3. Run locally with:

```bash
python3 assignments/games-in-python/starter-code.py
```

If your implementation is in a differently named file, run that file instead.

## ✅ Grading / Checklist

- Implements all requirements listed above
- Clean, readable code with sensible variable names
- Handles invalid input (non-letters, empty input) gracefully
- Optional stretch: add a list of categories, difficulty levels, or ASCII art for the hangman
